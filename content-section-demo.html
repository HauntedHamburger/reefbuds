Content Section Component Demo

This page demonstrates the different alignment options for the content section component.

Left-Aligned Content Section

Left-Aligned Heading

This content section is left-aligned, which is ideal for most reading scenarios. The text flows naturally from left to right, making it easy to read and follow.

Center-Aligned Content Section

Center-Aligned Heading

This content section is center-aligned, which is great for headlines, introductions, and other content that needs to be emphasized. It creates a balanced, symmetrical look.

Right-Aligned Content Section

Right-Aligned Heading

This content section is right-aligned, which can be used for specific design purposes or to create visual interest. Note that on mobile devices, this will revert to left-aligned for better readability.
